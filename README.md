Lakukan inisialisasi repository menggunakan git (CLI) dan buat clonenya di Github. Lalu, lakukan first commit berupa file README.md dan isinya tugas laporan ini. Dokumentasikan setiap langkah dan jelaskan tiap langkah dalam bentuk laporan

1. Download aplikasi git melalui link.
2. Kemudian pilih sesuai dengan ukuran laptop, saya memilih 64-bit Git for Windows Setup.
3. Tunggu hingga aplikasi berhasil diunduh.
4. Setelah selesai terunduh buka aplikasi, dan lakukan penginstallan.
5. Buka aplikasi Git Bash.
6. Lakukan lakukan konfigurasi identitas awal pada Git Bash.
7. Kemudian buka Github melalui google, kemudian jika sudah memiliki akun langsung klik sign in.
8. Kemudian buat repository baru.
9. Kemudian buka folder yang anda pilih untuk membuat folder baru.
10. Copy https Github dari repository kita di Github.
11. Kemudian membuat file README.md yang berisi laporan tugas ini.
12. Tempel https yang sudah di copy dari Github ke dalam Git Bash dengan Git clone.
13. Tambahkan file README.md ke dalam Git Bash dengan Git Add.
14. Lakukan first commit pada README.md di Git Bash menggunakan command Git Commit –m “first commit”.
15. Kemudian cek log menggunakan command Git Log.
16. Kemudian tambahkan command Git Push – u origin main untuk mengirim commit ke repository Github.
